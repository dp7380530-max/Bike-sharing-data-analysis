This project analyzes the Bike Sharing dataset using Microsoft Excel to identify key factors affecting bike rental demand. The analysis includes data cleaning, pivot tables, and interactive dashboard creation to visualize usage patterns based on season, weather, and time.
Key Features
•	Data cleaning and preparation in Excel
•	Pivot tables for trend and KPI analysis
•	Interactive dashboard with filters and charts
•	Insights on season, weather, and user behavior
Tools Used
Microsoft Excel
Outcome
Delivered a comprehensive Excel dashboard that visualizes bike usage trends and supports data-driven insights for better planning and decision-making.
